Navegadores web

Un navegador web (en ingles, web browser) es un software, aplicacion o programa que permite el acceso a la Web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad basica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari:Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS. Mozilla Firefox es un navegador web libre y de código abierto desarrollado para Linux, Android, iOS, macOS y Microsoft Windows coordinado por la Corporación Mozilla y la Fundación Mozilla. Google Chrome: Google Chrome es un navegador web de código cerrado desarrolado por Google, aunque derivado de proyecto de código abierto
